Border:

Agrega bordes al texto como en este ejemplo.

Border-left, Border-right, Border-top, Border-bottom:

Igualmente agrega bordes, pero en el lugar que se especifique

izquierdo

derecho

alto

bajo

Border radius:

agrega curvas a las esquinas de los bordes

Background Image:

agrega imagen en el fondo

Color:

Cambia el color del texto

Font:

cambia el estilo del texto

Font-Family:

Especifica el tipo de texto de un elemento

Font-style, Font-weight:

Igualmente cambia el estilo del texto, font style especifica si es italicas y font weight especifica si es bold y que tan anchas son las letras

Width, Height:

Define lo ancho y alto del elemento

List-Style:

Cambia el estilo de una lista, ejemplo:

  • hola 1
  • hola 2
  • hola 3

Margin:

Cambia el margen de un elemento

Padding:

Cambia el padding que es la parte interior de un elemento

Overflow:

Define que se muestra cuando el contenido no cabe dentro del elemento

Display:

Define como se mostratra el contenido del elemento con las opciones "none", "inline", "block", "inline-block"

Position:

Cambia el como se posiciona el elemento.

Filter:

Da varias opciones para cambiar el elemento, como el contraste o transparencia
